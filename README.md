# 소셜그로브 (Social Grove) 웹사이트

> 의미 있는 모임이 모여, 따뜻한 변화를 만든다 · 작은 그로브가 모여 의미 있는 숲이 된다.

balmoralrunning.com의 풀스크린 히어로 + 섹션 스토리텔링 구조를 참고해 만든
참여형 소셜 브랜드 사이트입니다. 20대 초중반이 보기에 재밌고 신뢰감 있도록
손그림 무드 + 그린/세이지 톤으로 디자인했어요.

## 실행
브라우저로 `index.html`을 더블클릭해서 바로 열면 됩니다. (별도 서버 불필요)

## 파일 구조
- `index.html` — 페이지 구조 + 콘텐츠
- `styles.css` — 디자인/반응형
- `script.js` — 네비/모바일메뉴/스크롤 애니메이션/카운터/신청폼
- `assets/` — 로고·사진 등 이미지 폴더

## 메뉴(섹션)
About us / 비전 / 가치 / 프로그램 / 아카이브 + 참여 신청(Join)

## 핵심 메시지 (요청 반영)
- **수익 일부 기부**: 가치 섹션 + 기부 배너에 강조
- **누구나 호스트(비재능자도 OK)**: 비전 섹션 + 프로그램 하단 CTA에 강조
- 나노 커뮤니티 / 함께 만들고·이야기하고·연결되는 경험 콘셉트

## 로고/사진 교체
- 현재 로고는 코드로 그린 **인라인 SVG**라 이미지 없이도 보입니다.
- 실제 로고 PNG를 쓰려면 `assets/logo.png`를 넣고, `index.html`의
  `<svg ...><use href="#grove-mark"/></svg>` 부분을 `<img src="assets/logo.png" ... />`로 교체하세요.
- 프로그램/아카이브/본문 사진은 `assets/*.jpg`에 들어 있어요. 실제 모임 사진으로 같은 파일명에 덮어쓰면 그대로 반영됩니다.

## 사진 라이선스
- `assets/`의 사진은 **Unsplash**(unsplash.com)에서 받은 무료 이미지입니다.
- Unsplash 라이선스: 상업적 사용 가능, 출처 표기 의무 없음(표기 권장). 사진 자체를 재판매하거나 경쟁 서비스로 배포하는 것만 금지.
- 핀터레스트 이미지는 대부분 저작권이 있어 사용하지 않았습니다. 실제 우리 모임 사진으로 교체하는 것을 가장 추천해요.

## 커스터마이즈 팁
- 색상: `styles.css` 상단 `:root` 변수만 바꾸면 전체 톤 변경
- 폰트: 손글씨 느낌은 `Gaegu` / `Nanum Pen Script`(Google Fonts) 사용
- 신청폼은 현재 데모(프론트 only)예요. 실제 수집은 Google Form/Tally/메일 연동 필요.
